webapp-minifier:minify-webapp

Full name:

com.github.lonnyj:webapp-minifier-maven-plugin:0.11.0-SNAPSHOT:minify-webapp

Description:

Minifies the web application.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: prepare-package.

Required Parameters

Name Type Since Description
encoding String 1.0 The character encoding used for HTML, JavScript and CSS files.
Default value is: ${project.build.sourceEncoding}.
minifiedDirectory File 1.0 The directory where the minified web application is built.
Default value is: ${project.build.directory}/${project.build.finalName}-minified.
sourceDirectory File 1.0 The web application source directory.
Default value is: ${basedir}/src/main/webapp.

Optional Parameters

Name Type Since Description
closureCompilationLevelString String 1.0 The Google Closure compilation level:
  • WHITESPACE_ONLY - The WHITESPACE_ONLY compilation level removes comments from your code and also removes line breaks, unnecessary spaces, extraneous punctuation (such as parentheses and semicolons), and other whitespace.
  • SIMPLE_OPTIMIZATIONS - The SIMPLE_OPTIMIZATIONS compilation level performs the same whitespace and comment removal as WHITESPACE_ONLY, but it also performs optimizations within expressions and functions, including renaming local variables and function parameters to shorter names.
  • ADVANCED_OPTIMIZATIONS - The ADVANCED_OPTIMIZATIONS compilation level performs the same transformations as SIMPLE_OPTIMIZATIONS, but adds a variety of more aggressive global transformations to achieve the highest compression of all three levels.
NOTE: This option can be overridden inline.
Default value is: SIMPLE_OPTIMIZATIONS.
User property is: closureCompilationLevel.
cssPrefix String 1.0 The prefix for minified CSS files.
Default value is: css.
htmlExcludes String[] 1.0 The HTML files to exclude from processing.
htmlIncludes String[] 1.0 The HTML files to include in processing.
Default values are:
  • **/*.html
  • **/*.htm
  • **/*.jsp
jsCompressorEngineString String 1.0 The JavaScript Compressor to use:
  • CLOSURE - The Google Closure Compiler.
  • YUI - The YUI Compressor.
NOTE: This option can be overridden inline.
Default value is: YUI.
User property is: jsCompressorEngine.
jsPrefix String 1.0 The prefix for minified JavaScript files.
Default value is: js.
mergeEmbeddedCss boolean 1.0 Merges embedded CSS with externally minified CSS.

NOTE: This option can be overridden inline.
Default value is: false.
mergeEmbeddedJs boolean 1.0 Merges embedded JavaScript with externally minified JavaScript.

NOTE: This option can be overridden inline.

For example,

    <name>/common</name>
    <value>${basedir}/../../common-project/src/main/webapp</value>
