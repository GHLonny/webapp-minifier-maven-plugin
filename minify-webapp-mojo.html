webapp-minifier:minify-webapp

Full name:

com.lonnyjacobson:webapp-minifier-maven-plugin:0.9.0-SNAPSHOT:minify-webapp

Description:

Minifies the web application.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: prepare-package.

Required Parameters

Name Type Since Description
encoding String 1.0 The character encoding used for HTML, JavScript and CSS files.
Default value is: ${project.build.sourceEncoding}.
sourceDirectory File 1.0 The web application source directory.
Default value is: ${basedir}/src/main/webapp.
targetDirectory File 1.0 The web application target directory.
Default value is: ${project.build.directory}/${project.build.finalName}-minified.

Optional Parameters

Name Type Since Description
closureCompilationLevel CompilationLevel 1.0 The Google Closure compiler level.

NOTE: This option can be overridden inline.
Default value is: SIMPLE_OPTIMIZATIONS.
cssPrefix String 1.0 The prefix for minified CSS files.
Default value is: css.
htmlExcludes String[] 1.0 The HTML files to exclude from processing.
htmlIncludes String[] 1.0 The HTML files to include in processing.
Default values are:
  • **/*.html
  • **/*.htm
  • **/*.jsp
jsCompressorEngine JavaScriptCompressor 1.0 The JavaScript Compressor to use:
  • CLOSURE - The Google Closure Compiler.
  • YUI - The YUI Compressor.
NOTE: This option can be overridden inline.
Default value is: YUI.
jsPrefix String 1.0 The prefix for minified JavaScript files.
Default value is: js.
mergeEmbeddedCss boolean 1.0 Merges embedded CSS with externally minified CSS.

NOTE: This option can be overridden inline.
Default value is: false.
mergeEmbeddedJs boolean 1.0 Merges embedded JavaScript with externally minified JavaScript.

NOTE: This option can be overridden inline.
Default value is: false.
otherDirectories Properties 1.0 Defines other directories where CSS and JavaScript files may be found. This is useful when other projects contain common CSS and JavaScript. The name should be a location as defined in the HTML. The value would be the directory that corresponds to the location.

For example, 
<otherDirectories>
  <property>
    <name>/common</name>
    <value>${basedir}/../../common-project/src/main/webapp</value>
  </property>
</otherDirectories>

skipCssMinify boolean 1.0 Skips CSS minification.

NOTE: This option can be overridden inline.
Default value is: false.
skipEmbeddedCssMinify boolean 1.0 Skips Embedded CSS minification.

NOTE: This option can be overridden inline.
Default value is: false.
skipEmbeddedJsMinify boolean 1.0 Skips Embedded JavaScript minification.

NOTE: This option can be overridden inline.
Default value is: false.
skipJsMinify boolean 1.0 Skips JavaScript minification.

NOTE: This option can be overridden inline.
Default value is: false.
skipMinify boolean 1.0 Skips any minification processing.
Default value is: false.
yuiCssLineBreak int 1.0 Instructs the YUI Compressor to break lines after the specified number of characters.
  • -1 - Disables line breaking.
  • 0 - Causes a line break after each rule in CSS.
NOTE: This option can be overridden inline.
Default value is: -1.
yuiJsDisableOptimizations boolean 1.0 Instructs the YUI Compressor to disable all JavaScript micro-optimizations.

NOTE: This option can be overridden inline.
Default value is: false.
yuiJsLineBreak int 1.0 Instructs the YUI Compressor to break lines after the specified number of characters.
  • -1 - Disables line breaking.
  • 0 - Causes a line break after each semi-colon in JavaScript.
NOTE: This option can be overridden inline.
Default value is: -1.
yuiJsNoMunge boolean 1.0 Instructs the YUI Compressor to only minify JavaScript without obfuscating local symbols.

NOTE: This option can be overridden inline.
Default value is: false.
yuiJsPreserveAllSemiColons boolean 1.0 Instructs the YUI Compressor to preserve unnecessary semicolons in JavaScript. This option is useful when compressed code has to be run through JSLint.

NOTE: This option can be overridden inline.
Default value is: false.
