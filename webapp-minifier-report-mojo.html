webapp-minifier:webapp-minifier-report

Note:This goal should be used as a Maven report.

Full name:

com.lonnyjacobson:webapp-minifier-maven-plugin:0.9.0-SNAPSHOT:webapp-minifier-report

Description:

Generates a report summarizing the minification performed on the web application.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: site.

Required Parameters

Name Type Since Description
outputDirectory File - The report output directory. Note that this parameter is only evaluated if the goal is run directly from the command line or during the default lifecycle. If the goal is run indirectly as part of a site generation, the output directory configured in the Maven Site Plugin is used instead.
Default value is: ${project.reporting.outputDirectory.
