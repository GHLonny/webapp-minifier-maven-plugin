Configuring Inline

Introduction

The Webapp Minifier plugin allows for some configuration to be overridden inside the HTML by leveraging comment tags. These comment tags must include the option override header "webapp-minifier-maven-plugin:". The configuration is parsed like Java properties and remains in affect for the duration of the file. The next file will start with the original plugin's configuration.

Configuration

For single parameter changes, you can leave it all on the same line like this:

<!-- webapp-minifier-maven-plugin: skipCssMinify=true -->

For multiple parameter changes, you must break them up into separate lines:

<!--
webapp-minifier-maven-plugin: 
jsCompressorEngine=CLOSURE
closureCompilationLevel=ADVANCED_OPTIMIZATIONS
-->

Commenting

Any comments may appear prior to the option override header.

<!--
Break after every 1000 characters.
webapp-minifier-maven-plugin: yuiCssLineBreak=1000
-->

After the option override header, the remainder of the tag's content is parsed using Java's Properties class. A comment line starts with either a '#' or '!'.

<!--
webapp-minifier-maven-plugin:
! Switch to Google's Closure compiler.
jsCompressorEngine=CLOSURE
# The remaining JS compresses best with the advanced optimizations.
closureCompilationLevel=ADVANCED_OPTIMIZATIONS
-->
